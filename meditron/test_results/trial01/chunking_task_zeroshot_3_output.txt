Inclusion Criteria
•	Male or female patients age ≥ 18 years
•	Patients must have histologically confirmed HER2+ metastatic breast cancer
•	Patients must have measurable disease according to RECIST v1.1
•	ECOG performance status of 0 or 1
•	Life expectancy > 6 months
•	Adequate hematologic function defined as absolute neutrophil count ≥ 1500 cells/mm3, platelet count ≥ 100,000 cells/mm3, and hemoglobin level ≥ 9 gm/dl
•	Adequate hepatic function defined as total bilirubin ≤ 1.5 x ULN, ALT ≤ 2.5 x ULN, and AST ≤ 2.5 x ULN
•	Adequate renal function defined as creatinine clearance ≥ 60 ml/min
